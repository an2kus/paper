Правила формирования CSV-файла

1. Формировать таблицу с данными нужно в Excel
2. В первой строке таблицы должны быть названия полей латиницей согласно Регламента
3. Сфорированную таблицу нужно сохранить в формате CSV
4. При сохранении необходимо указать разделитель полей {Tab} и разделитель строк \n
5. Файлу следует присвоить имя list_students.csv
6. Готовый CSV-файла нужно закачать на сервер.
7. Если все сделано правильно, на сервере будет сформирован XML-файл, готовый к скачиванию.

ПОСЛЕ ПОЛУЧЕНИЯ ФАЙЛА С СЕРВЕРА
ЕГО НУЖНО РАСПАКОВАТЬ И ПЕРЕИМЕНОВАТЬ

Имена файлов должны иметь следующий формат:
ККК-YYMMDD-XXX-NNNNN.XML, где:
ККК – код ВУЗа по НСИ ГУП МСР (код МГИМО: 186);
YYMMDD – год месяц и дата создания файла;
XXX – порядковый номер файла за указанную дату;
NNNNN– количество записей в выгрузке.

Отладка

  • Если в xml-файле нет полей, которые есть в csv-файле, нужно править файл иерархии.
    Для этого:
    • Откройте файл иерархии (он откроется в новом окне/вкладке браузера);
    • Скопируйте текст в редактор (лучше всего в Блокнот) и добавьте новые теги в нужное место;
    • Сохраните текст в файл под именем hierarchy (без расширения);
    • Закачайте файл иерархии на сервер.

  • Если в csv-файле нет полей, которые есть в xml-файле, нужно править ваш csv-файл.
    Для этого:
    • Откройте в Экселе свой csv-файл и отредактируйте его в соответствии с файлом иерархии;
    • Сохраните свой csv-файл под именем list_students.csv;
    • Закачайте свой csv-файл на сервер.
